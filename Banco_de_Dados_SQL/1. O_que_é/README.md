# 🗄️ O que é um Banco de Dados

---

## 1️⃣ Evolução do armazenamento de dados

### 📜 Antigamente
- Dados armazenados em **papel**
- Difícil acesso e organização

---

### 💾 Evolução digital
- Arquivos sequenciais
  - Dados armazenados em ordem
  - Acesso lento

---

### ⚡ Acesso direto
- Uso de **índices**
- Permite encontrar dados rapidamente
- Base para sistemas modernos

---

## 2️⃣ Composição de um Banco de Dados

Um banco de dados é formado por:

### 📦 Base de dados
- Onde os dados são armazenados
- Ex: tabelas, registros, arquivos

---

### ⚙️ Sistema Gerenciador (SGBD)
- Software que controla o banco
- Responsável por:
  - Inserir dados
  - Atualizar
  - Deletar
  - Garantir segurança

---

### 🔍 Linguagem de exploração
- Linguagem usada para acessar os dados
- Ex: consultas, filtros

---

### 🧩 Programas adicionais
- Interfaces e sistemas que utilizam o banco
- Ex: sistemas web, apps, ERPs

---

## 3️⃣ Modelos de Banco de Dados

### 🌳 Hierárquico
- Estrutura em árvore
- Relação pai → filho

---

### 🔗 Em rede
- Permite múltiplas conexões
- Mais flexível que o hierárquico

---

### 🧱 Relacional (mais usado)
- Baseado em **tabelas**
- Dados organizados em:
  - Linhas (registros)
  - Colunas (atributos)

💡 É o modelo padrão atual

---

## 4️⃣ SQL

### 🗣️ O que é
- Linguagem usada para interagir com bancos relacionais

### 📌 Para que serve
- Consultar dados
- Inserir
- Atualizar
- Remover

---

## ✔ Conclusão

- Banco de dados organiza e armazena informações  
- Evoluiu de papel → arquivos → sistemas modernos  
- O modelo relacional é o mais utilizado  
- SQL é a principal linguagem para manipulação de dados
